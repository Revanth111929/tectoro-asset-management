## 🛠️ Helper Scripts

### check_requirements.sh
**Purpose**: Verify system requirements  
**Run this**: Before setup  
**What it does**:
- Checks if Python installed
- Checks if Node.js installed
- Checks if npm installed
- Shows your IP address
- Tells you what's missing

**Usage**:
```bash
./check_requirements.sh
```

---

### START_HERE.sh
**Purpose**: Automated setup  
**Run this**: After checking requirements  
**What it does**:
- Creates Python virtual environment
- Installs Python dependencies
- Installs Node.js dependencies
- Configures API URL with your IP
- Shows next steps

**Usage**:
```bash
./START_HERE.sh
```

## 📞 Getting Help

### Step 1: Check Documentation
Look for your issue in:
1. TROUBLESHOOTING.md
2. README.md
3. SETUP_GUIDE.md

### Step 2: Check Logs
- Browser console (F12)
- Terminal 1 output (backend)
- Terminal 2 output (frontend)

### Step 3: Search Online
- Copy error message
- Search on Google
- Check Stack Overflow

### Step 4: Debug
- Read error message carefully
- Check what changed recently
- Try restarting servers
- Clear browser cache
